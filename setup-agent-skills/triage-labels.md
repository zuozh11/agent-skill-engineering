# 分诊标签

Skill 使用四个规范的分诊角色。本文件将这些角色映射到本仓库 task 追踪器中实际使用的标签字符串。

| Skill 中的角色        | 本仓库标签             | 含义                  |
|-------------------|-------------------|---------------------|
| `needs-triage`    | `needs-triage`    | 维护者需要评估此 task       |
| `needs-info`      | `needs-info`      | 等待提报人补充信息           |
| `ready-for-agent` | `ready-for-agent` | 描述完整，AI Agent 可独立处理 |
| `ready-for-human` | `ready-for-human` | 需要人类开发者实施           |

当 skill 提到某个角色（如"应用 AFK-ready 分诊标签"），使用本表中对应的标签字符串。

如需自定义，编辑右侧列即可。
